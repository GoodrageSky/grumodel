# 研报文本因子流水线 · 模型构建与实验进展报告

> **日期**：2026-06-01

**一句话摘要**：端到端流水线已跑通。经系统诊断，最初"样本外 IC 为负"的根因是**月频评估无统计功效**（非模型/符号错误），改周/日频后强因子显著。在恢复功效的评估上穷尽检验研报文本的各种利用形式（嵌入 / 覆盖度 / 情感水平 / 情感序列 / LLM 正文观点），结论是：**研报文本仅能提供一个弱情感标量因子（Sharpe≈1.2），对量价组合无增量**；覆盖度本质是规模代理，正文缺乏结构化评级/目标价信号。诊断中意外得到一个**可投产的纯量价组合（Sharpe 2.54）**。下一步：量价组合投产化 + 文本结论归档；文本路线重启取决于能否取到带评级字段的研报库。

---

## 1. 项目目标与模型架构

**目标**：把 A股研报文本转化为可用于选股的跨截面因子，并以严格的样本外（OOS）口径评估其有效性（IC / RankIC / ICIR / 多空夏普）。

**四阶段流水线**：

| 阶段 | 模块 | 职责 |
|---|---|---|
| Stage 1a/1b | 文本日频化 + 摘要 | 研报库 → (交易日,股票) 去重聚合 → 300-500字摘要 |
| Stage 2 | 嵌入 + 有监督降维 | FinBERT(bert-base-chinese) 768维 → 仅train拟合的有监督UMAP → 10维 |
| Stage 3 | GRU + LGBM 因子 | GRU序列特征 → LGBM → 截面因子 + IC/多空回测 |
| 对照 | 全文 vs 摘要 | 闭环验证摘要是否丢失 alpha |

**数据口径（关键）**：标签源 `dt_lgbm.parquet` 为**月频**，面板共 68 个月度截面、372 只研报覆盖股，每截面平均仅约 116 只有文本。

---

## 2. 实验过程（按时间顺序）

### 2.1 基线流水线跑通

完成 Stage1→3 全链路。文本压成 10 维特征后喂入 LGBM，并与"裸 4096 维嵌入直喂 Stage3"的快速通道对比。

### 2.2 直接 LGBM 基线（direct_lgbm）实测结果

| 数据段 | IC_mean | RankIC_mean | ICIR | 截面数 | t 值 |
|---|---|---|---|---|---|
| train | +0.1956 | +0.1906 | +1.598 | 35 | +9.45 |
| valid | -0.0154 | -0.0288 | -0.192 | 14 | -0.72 |
| test | +0.0161 | +0.0191 | +0.213 | 17 | +0.88 |

**解读**：train 强正（t=9.45）但 valid/test 的 |t|<2，与 0 无差异。train 的高 IC 是"有监督降维用未来收益分位拟合 train 标签"的产物，不代表泛化能力。
### 2.3 滚动窗口（cons_roll）与全文/摘要对照

| 实验 | test IC | test ICIR | 多空夏普 |
|---|---|---|---|
| cons_roll_2023_04 | -0.0226 | -0.220 | +0.14 |
| cons_roll_2023_10 | -0.0273 | -0.264 | -0.46 |
| cons_roll_2024_04 | +0.0066 | +0.089 | +1.75 |
| full_text vs summary | -0.017 / -0.015 | -0.166 / -0.141 | — |

**解读**：test IC 符号在窗口间随机翻转；full_text 与 summary 两种文本口径 OOS 都 ≈ -0.015。这排除了"符号学反"（train 强正），指向 OOS 根本无可辨识信号。

### 2.4 受控诊断：信号到底在哪一层丢失（diag_signal）

用与真实流水线相同的 purged 切分 + IC 计算代码，逐层消融：

| 实验 | valid IC (t) | test IC (t) | 判定 |
|---|---|---|---|
| 正控制（标签+噪声） | +0.995 (t=2760) | +0.994 (t=1812) | 评估/对齐代码正确 |
| 负控制（纯噪声） | -0.018 (-0.63) | +0.026 (+0.89) | 噪声基准 |
| raw 768维 + 岭回归 | +0.004 (0.11) | -0.001 (-0.02) | 零信号 |
| 无监督 PCA 10维 | -0.032 (-1.06) | +0.048 (+1.09) | 零信号 |
| 有监督 UMAP 10维 | -0.005 (-0.13) | +0.060 (+2.00) | ≈噪声 |

**关键结论**：
1. 正控制 t≈千级 → 评估/对齐/purge 代码无误，负 IC 是真实数据而非 bug；
2. 任何文本表示（原始768/PCA/有监督UMAP）OOS |t| 均 <2，与纯噪声无法区分 → 不是降维/调参问题；
3. 有监督 UMAP 的 OOS 并不优于原始，印证 train IC 是泄漏式拟合产物。

### 2.5 天花板测试：评估窗口有没有统计功效（diag_ceiling）

引入平台量价数据，用公认强因子检验当前月频窗口能否测出任何因子。（test 段 17 个月度截面，372 股）

| 因子 | test IC (t) | 因子 | test IC (t) |
|---|---|---|---|
| 动量 mom_12_1 | -0.0155 (t=-0.32) | 流动性 liq_amt20 | -0.0310 (t=-1.02) |
| 反转 rev_1m | +0.0153 (t=0.30) | 换手率 turn_20 | -0.0387 (t=-1.07) |
| 波动率 vol_60 | -0.0278 (t=-0.45) | 小市值 size | +0.0158 (t=0.36) |

**决定性结论**：连换手率、小市值、波动率这些 A股最强因子，在该月频窗口里 |t| 全部 <2 —— 评估窗口本身没有分辨力。之前文本因子的"负 IC"只是这个无功效窗口里的噪声。

### 2.6 周频验证：改频率能否恢复功效（diag_weekly_ceiling）

用平台日频价格自建周频截面（test 段 70 个截面）+ 未来1周收益标签，重跑同样因子：

| 因子 | 月频 test t | 周频 test t（全市场） | 是否显著 |
|---|---|---|---|
| 流动性 liq_amt20 | -1.02 | -3.09 | ✓ 显著 |
| 小市值 size | +0.36 | +2.29 | ✓ 显著 |
| 反转 rev_1w | — | +1.65 | 接近 |
| 换手率 turn_20 | -1.07 | -1.20 | 改善 |

**验证成立**：截面数 17→70 后，多个强因子 |t| 升破 2，符号在 train/valid/test 三段一致。证明瓶颈是评估频率太粗，t 值随 √截面数 放大的规律完全兑现。

已据此构建周频标签源 `weekly_labels.parquet`（326 个周度截面，未来1周 vwap 收益 + 横截面分位）。

### 2.7 日频天花板：功效拉满 + 暴露"信号衰减"陷阱（diag_daily_ceiling）

进一步推到日频（test 325 个截面），用 Newey-West 校正重叠收益的 t 值：

| 因子 | 月频 t | 周频 t | 日频 t (fwd_1d) | 日频 t (fwd_5d NW) |
|---|---|---|---|---|
| 流动性 liq_amt20 | -1.02 | -3.09 | -4.16 | -4.47 |
| 小市值 size | +0.36 | +2.29 | +2.74 | +2.75 |
| 反转 rev_1d | — | — | -4.91 | +0.41 |

**两个发现**：①强因子 t 值随频率单调放大（liq: -1.0→-3.1→-4.5），评估功效彻底确认；②反转因子 fwd_1d 极强(t=-4.9)但 fwd_5d 归零(+0.4) —— **同一因子换持仓周期信号即消失**，提示"频率对了、持仓周期错了照样测不到"，研报信号也需先确定衰减曲线。

### 2.8 研报到达事件研究 + 市值中性化（diag_report_event / diag_neutralize）

绕开"被 snap 的月度日期"，直连 RDS 拉**真实发布日**的每股每日研报计数，测"研报到达"对未来收益的 IC：

| | raw IC (fwd_20) | 市值中性化后 |
|---|---|---|
| 研报到达 has_report | -0.0222 (t=-2.2) | **-0.0004 (t=-0.1)** |

**关键结论**：研报到达的"负向长周期效应"市值中性化后**归零** —— "研报覆盖度/数量"本质是**规模(大市值)代理**，是 beta 不是 alpha。这解释了最初文本因子为何 IC 为负：研报多的票=大盘拥挤股，后续跑输。

### 2.9 研报观点方向因子：标题情感 vs 摘要情感

转向"研报说了什么"（观点方向），而非"有没有研报"（覆盖度）。市值中性化残差上测：

| 因子 | train t | valid t | test t | walk-fwd t | 多空 Sharpe |
|---|---|---|---|---|---|
| 标题情感 sent_avg | 1.84 | 0.53 | 1.23 | 2.24 | **0.92** |
| 摘要情感（扩词典+评级词） | **5.31** | -0.68 | 0.92 | 3.81 | 0.26 |

**结论**：①方向对了 —— 从"数量"换成"观点方向"，信号由噪声变正向；②但**偏弱**：标题版 Sharpe 仅 0.92，valid 段塌陷；③摘要版扩词典后 **train 飙到 t=5.31 但 OOS 更差**（Sharpe 0.26）—— **过拟合**，证明 A股研报情感的真实 OOS 上限就在 Sharpe≈1。标题版（简单、不易过拟合）反而最稳。

### 2.10 GRU 前置验证：情感的时间序列结构无增量（diag_senti_sequence）

回归项目本意（GRU 建模研报情感序列）前，先验证必要条件——序列结构是否比单期快照多带信息：

| 特征 | test t | walk-fwd t |
|---|---|---|
| senti_now（单期快照·基线） | 2.36 | 3.42 |
| senti_chg（一阶变化） | 0.61 | 0.88 |
| senti_ma3（平滑） | 2.09 | 3.41 |
| senti_trend（趋势） | 1.77 | -0.08 |

**结论**：所有"变化型"特征都**不优于**单期快照，平滑(ma3)只是降噪、t 持平。信息全集中在"当前情感水平"这一个标量上 —— **序列结构无增量，GRU 必过拟合，不该训**。5 分钟前置检验省下训练成本。

### 2.11 LLM 正文结构化观点抽取（diag_llm_opinion）

唯一未穷尽的路径：用 DeepSeek 从 4000 篇研报**正文**抽 {评级、评级变化、目标价隐含涨幅、盈利调整}。抽取成功率 3997/3999，但质检发现**信号源本身极度稀疏**：

| 字段 | 有效信息占比 |
|---|---|
| 评级变化（上调/下调） | **0.75%**（30/3999） |
| 目标价隐含涨幅 | **4%**（167/3999） |
| 盈利预测调整方向 | 6.5%（258/3999） |

**结论**：不是 LLM 没抽出来，而是 A股研报正文**不是结论驱动**的，明确的评级变化/目标价极少出现。扩样本也救不回 0.75% 的覆盖率 —— 此数据源下 LLM 观点因子**无法构成截面因子**。

### 2.12 多因子合成：文本对量价无增量（factor_combo）

把标题情感与 5 个量价因子（反转/小市值/流动性/动量/低波）等权合成，市值中性化残差上：

| 组合 | walk-fwd t | test Sharpe |
|---|---|---|
| 文本单因子 senti | — | 1.23 |
| **纯量价合成（5因子）** | **3.83** | **2.54** |
| 全合成（+文本） | 3.67 | 2.47 |

**结论**：尽管文本与量价低相关（|ρ|<0.1），但因自身太弱，加入后 Sharpe **从 2.54 降到 2.47** —— **文本对量价组合无增量、略有拖累**。项目的真正可投产产出是**纯量价合成组合（Sharpe 2.54）**，与研报文本无关。

## 3. 核心结论：研报文本因子的路径已穷尽

围绕"如何用研报文本优化因子"这一主课题，所有可行利用形式均经受控实验检验：

| 路径 | 方法 | 结论 |
|---|---|---|
| 无监督嵌入 | bert-base-chinese 768维 / UMAP / PCA | OOS \|t\|<2，与噪声无异 |
| 覆盖度 / 数量 | 研报到达计数 | 市值中性化后归零 —— 是规模 beta，非 alpha |
| 情感水平 | 标题 / 摘要词典情感 | 弱（Sharpe~0.9），摘要扩词典即过拟合 |
| 情感序列结构 | 变化 / 趋势 / 加速度 / 意外 | 不优于单期快照 —— GRU 序列建模不成立 |
| LLM 正文观点 | DeepSeek 抽评级/目标价变化 | 信号源稀疏（评级变化0.75%/目标价4%），无法成因子 |

**最终判断**：研报文本里唯一能稳定提取的，是"当前情感水平"这一**弱标量因子**（test t≈2.4，多空 Sharpe≈1.2），且它对量价组合**无增量**。在当前数据源（RDS report_info：标题/摘要/正文，无评级/目标价/盈利预测结构化字段）下，研报文本的 alpha 上限很低且无法突破。

**意外收获**：诊断过程中构建的**纯量价合成组合**（反转/小市值/流动性/动量/低波，等权，市值中性化）walk-forward t=3.83、test 多空 Sharpe=2.54、胜率 70%，三段 IC 同号，是一个真正可投产的组合 —— 与研报文本无关。

## 4. 下一步方案

| 优先级 | 动作 | 说明 |
|---|---|---|
| P1 收口（推荐） | 量价组合投产化 | 纯量价组合(Sharpe 2.54)做行业中性化 + 完整回测（净值/回撤/换手/分年度），投产 |
| P1 归档 | 文本结论固化 | 标题情感作为"已验证、增量有限"的弱卫星因子归档 |
| P2 探索 | 换研报数据源 | 当前 RDS 库正文非结论驱动；若公司有**带评级/目标价的一致预期库**（朝阳永续/Wind con），评级变化因子才有原料，值得重启文本路线 |
| P3 可选 | 文本作小权重卫星 | 标题情感与量价低相关，用 IC 加权（非等权）尝试挤出微弱增量 |

**关键决策点**：研报文本课题在现有数据源下已研究透彻。是否重启取决于能否取到**带结构化评级/目标价字段的研报库** —— 这是海外"研报 alpha"的标准数据源，也是本数据源缺失的关键。
